### 为什么会有工具系列专题

好比, 初入江湖, 要想成为一个武林高手。必须要有一个基础的内功心法。内功心法越高, 练成最上乘武功就越快。
而这些常用的框架，就是基础的内功心法。当然没有内功心法也能练成上乘武功。只不过可能时间花费会更多点。
本篇小编分享下常用的Java 工具。这些会在后面我们写框架或者是写业务代码时候。如果你还有更多好的工具,可以留下
你的评论。我们一起学习。
